# xboard-static-client

一个基于 XBoard API 构建的纯静态 HTML 用户端。

只有一个 css 和一个 js 和一堆 html ，通过 API 与 xboard 通信。部分内容由 AI 生成，采用 Gemini 3 Pro。纯粹的 HTML 设计，使得修改极其方便，不需要再为 vue 难以修改的成品头疼，你的个性化方法，可以随时落地执行。

· 该前端的 js 设计有 API 缓存，在得到响应之前先显示缓存数据，避免每次都要等加载。

· 已有功能：仪表盘、上手指南（教程）、流量明细、服务器状态（节点状态）、变更订阅（购买产品）、订单记录、收银台（支付）

**[ 注：账户管理和注册使用了独立的账户系统，你可以考虑自行合并。账户系统：等待公布。 ]**

## 如何部署？

### 全局：

· 在 vsc 打开文件夹，全局搜索“自定义名称”并替换为你的品牌名字。

· 你可以全局搜索“https”，快速查找包含链接的部分，快速进行修改。

### js：

· 最顶部填写你的 API 地址，不需要添加尾斜杠“/”

· 顶栏包含在 js 中，如果你想，你可以修改 js 中的顶栏部分实现自定义功能。这一部分为 HTML 。

· 版权信息可以去除，可以更换为你自己想要的信息。

### css：

· 基本无需修改，如果你想换个颜色，只需要调整变量即可。css按照字母顺序排序，方便定位查找。

### html：

· 除了收银台，其他地方几乎无需改动。我们采用了礼品卡兑换余额支付法，如果你也是这样，请修改“购买礼品卡的地址”。

· 如果你希望接入支付接口，你需要自行编辑页面。

## 局限：

· 该面板不再支持“公告”、“订阅详情”等 Markdown 文本，如果你需要特殊格式，比如加粗，请使用 b标签 而不是 **** 。

· 公告不支持背景图片显示，如果需要，请自行修改。

· 账户类管理功能不包含在面板内，登录页也不包含注册功能，我们会提供账户管理器的页面，如果你想，你可以合并到此面板中。

## 最后：

· 虽然功能日益完善，但这仍然是 beta 版本。

· 尚不能照顾所有人对面板的美好想象，不满意的地方请自行修改。

· HTML 的修改非常简单，如果你不会，也可以让 AI 辅助， HTML 的编辑绝大多数的 AI 都能做到，记得把 css 和 js 一起喂给 AI 。
